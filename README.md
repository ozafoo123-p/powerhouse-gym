# PowerHouse Gym — Business Website (Portfolio Project)

PowerHouse Gym is a fully responsive, single-page website built as a portfolio demo to showcase a complete small-business web presence — from first impression to conversion. It uses only HTML, CSS, and vanilla JavaScript, with no frameworks or backend, making it lightweight and fast to load.

**Key Sections:**
- **Hero Section** — Bold full-screen intro with background image, tagline, and call-to-action button linking to membership plans.
- **About** — Short section explaining the gym's mission and value.
- **Classes** — Card grid showcasing Strength Training, HIIT Cardio, and Yoga & Flexibility.
- **Membership Pricing** — Three tiers (Basic, Pro, Elite) for easy comparison.
- **Contact Form** — Functional inquiry form (front-end only, doesn't send real data).

**Design Approach:**
Dark, high-contrast theme with a bold orange accent for a fitness-brand feel. Built with Flexbox and CSS Grid for responsiveness across mobile, tablet, and desktop — no external frameworks.

**Note:** This is a demo/portfolio project to demonstrate front-end design and deployment skills. PowerHouse Gym is not a real business, and the contact form does not send or store data.
